# 🚀 office2pdf 转换器

**一款小巧、无需联网、永久免费的 Office 办公格式终极转换神器！**

如果你正在发愁如何把大量的合同、表格或者幻灯片变成排版不乱的 PDF，或者你受够了网上那些充满烦人广告、而且需要逐个上传下载的各种野生在线 PDF 转换网站，那么 **office2pdf** 就是为你量身打造的。

## 🎯 我们的四大绝对优势

### 📂 1. 万能黑洞：拖进来，什么格式都吃！
你可以把一堆 **Word (`.doc`, `.docx`)**、**Excel 表格 (`.xls`, `.xlsx`)**，还有 **PPT 幻灯片 (`.ppt`, `.pptx`)**，几十几百个全都杂在一起，闭着眼睛直接拖拽到这个黑色的窗口里，前端会自动给你分拣好类型，不用你再人工整理！

### ⚡ 2. 恐怖的转换速度
别家软件是慢吞吞的一个个打开 Word，再一个个关闭。我们的地盘我们做主！
软件采用了**全自动聚合批处理技术**：它能在毫秒间对你丢进去的所有文件进行分类打包分发！底层采用后台进程驻留，不论你是几十个表格还是上百个合同，只需启动一次常驻环境，便能瞬间行云流水般抽出完美的 PDF！

### 🤫 3. 百分百隐私安全：永远断网运行
所有的操作都在你自己的电脑上以绝对静默的方式本地运行，不会上传一丝一毫的数据到服务器。连转换时的后台黑框都不会出现，确保你的机密合同、敏感打卡表万无一失。

### 🗜️ 4. 极致微型体积
我们追求的是“单文件闯天下”。不同于市面上常见的动辄一两百兆的 Electron 打包软件，我们通过底层的原生框架（Rust + Tauri），硬生生地将体积压到了惊人的 **不足 10 MB**！不管是放在 U 盘还是塞进微信发给同事，都是瞬发的福音。

## 💻 快速开始

1. **下载使用**：前往本项目的release，下载我们为你准备好的独立版 `.exe`。
2. **免安装直通**：直接双击即用，拖入你要转换的文件，点击“一键极速转换”。
3. **查收成果**：所有完美导出版式的 PDF 都将无缝并乖巧地跟你原始文件呆在同一个文件夹里！

## 🛠️ 为开发者准备

如果你想参与二次开发或者体验本地最新的改动：

```bash
# 确保安装了 Rust 和 Node (v20以上)
npm install
npm run tauri dev      # 启动带热更新的调试版界面
npm run tauri build    # 输出供大家传阅分发的仅几 MB 的压缩 EXE 极简最终版！
```
