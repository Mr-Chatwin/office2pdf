# OfficeToPDF - 极速跨端文档排版转换器 ⚡

原本为单源的 PPT 处理工具，现已进行**底盘彻底重构**，脱胎换骨为搭载纯血 **Rust + Tauri v2** 核心引擎的“办公三大件”超级聚合平台。摒弃了所有沉重过时的 Web 和 Electron 包袱。

### ✨ 核心突破 

- **🗜️ 极致微缩，摆脱臃肿** 
  丢掉了上百兆的 Chromium (Electron) 和笨重的 Node 环境，单执行文件仅 **不到 8MB**，U 盘一拷，走到哪用到哪！
- **🕳️ 万能黑洞，智能吞吐**
  不再限制格式！支持把 `.doc`、`.docx`、`.xls`、`.xlsx`、`.ppt`、`.pptx` 几十上百个文件混搭在一起，一口气往转换框里砸，程序能智能分配对应引擎。
- **🚀 后台聚合，性能拉满**
  破除了传统转换器“来一个，开一次 Office”的龟速逻辑。底层采用极其强悍的 **异步并行引擎分发池 (`convert_batch`)**，只需极低的首发开销（仅 5 秒），剩余几百个文件全自动顺势被瞬间榨干成完美的 PDF！
- **🤫 隐形潜行，纯净无扰**
  抛弃了闪烁的 CMD 黑框，直接使用最底层的 `std::process::Command` 并绑定特权属性拦截 Windows CLI，完美实现了真正的后台深层静默剥离操作。 

### ⚙️ 使用说明

你**不再需要**在本地浪费算力与时间去构建。只要向仓库推送更新：

1. 打开本项目的 GitHub 官方 [**Actions**] 页面。
2. 找到最新的那次自动构建。
3. 直接拖到页面最底端，把 **OfficeToPDF-Win-Executable** (Exe 制品) 下载下来。
4. 双击运行，即可开始畅享原汁原味的、绝不掉帧跑版的办公转换旅程。
